Social Visualization with Behaviour Logging
Website which display behaviour logging by tracking user interactions on StackOverFlow website and captures data. This website displayes interactive social visualiization
and explains the explored patterns.

Getting Started
These instructions will get you copy of the project up and running on your local machine for testing.


Prerequisities
These are the softwares you need to install
MongoDB
NodeJS
Google Chrome


SetUp
Create - Database with the name "AdaptiveWeb" in mongodb
Go to Chrome Extension - Enable developer mode, Select Load Unpacked and upload the chrome extension "Chrome Extension for StackOverFlow"
Go to "Logging Behaviour" project folder
Run node server.js
